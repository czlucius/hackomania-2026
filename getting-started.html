Get Started in 5 Minutes

1

Generate Icons

First, create the required icon files for the extension:

# Navigate to project directory cd hackomania2026 # Generate icons using Python python3 create_icons.py
Alternative: If you don't have Python/PIL, run ./setup-icons.sh (requires ImageMagick) or manually add icon files to the images/ folder.
2

Load Extension in Chrome

Install the extension in your Chrome browser:

  • Open Chrome and navigate to chrome://extensions/
  • Toggle Developer mode (top-right corner)
  • Click Load unpacked button
  • Select the hackomania2026 folder
  • Verify the extension appears in your list
Chrome Extensions Page Screenshot

Extensions → Developer Mode → Load Unpacked

3

Open Telegram Web

Navigate to Telegram Web and log in:

  • Go to https://web.telegram.org
  • Log in with your phone number
  • Enter the verification code sent to your Telegram app
Important: This extension only works on Telegram Web, not the desktop or mobile apps.
4

Activate & Configure

Click the extension icon (🛡️) in your Chrome toolbar:

  • Verify status shows "Active" with a green dot
  • Keep Enable Protection toggled ON
  • Choose between Automatic or Manual mode
  • Optional: Add API keys for enhanced fact-checking
5

Start Using!

The extension is now monitoring your chats:

  • Open any Telegram chat
  • The extension automatically analyzes messages
  • Look for colored warning badges on suspicious messages
  • Click Details to see why a message was flagged
Success! You're now protected from fake news on Telegram Web. Messages will be analyzed automatically.

Complete Installation Guide

Prerequisites

  • Google Chrome version 88 or higher
  • Active Telegram account
  • Python 3.6+ (for icon generation) OR ImageMagick
  • Basic command line knowledge

Detailed Steps

1. Download the Project
# Clone from GitHub git clone https://github.com/yourusername/telegram-fake-news-checker.git cd telegram-fake-news-checker/hackomania2026 # Or download ZIP and extract
2. Create Icons (Choose One Method)

Metho

d A: Python Script (Recommended)

# Install PIL if needed pip3 install Pillow # Generate icons python3 create_icons.py

Method B: Bash Script

# Install ImageMagick if needed (Ubuntu/Debian) sudo apt-get install imagemagick # Run script chmod +x setup-icons.sh ./setup-icons.sh

Method C: Manual Download

Download shield icons from Flaticon or Icons8 and save as:

  • icon-16.png (16x16 pixels)
  • icon-32.png (32x32 pixels)
  • icon-48.png (48x48 pixels)
  • icon-128.png (128x128 pixels)
3. Load in Chrome
  1. Open Chrome browser
  2. Type chrome://extensions/ in address bar
  3. Enable "Developer mode" toggle (top-right)
  4. Click "Load unpacked" button
  5. Navigate to and select hackomania2026 folder
  6. Click "Select Folder" or "Open"
  7. Extension should appear in your list
  8. Pin to toolbar (click puzzle icon, then pin)
4. Configure Settings

Click the extension icon to access settings:

  • Enable Protection: Toggle on/off
  • Checking Mode:
    • Automatic: Checks all messages continuously
    • Manual: Check only when you click the button
  • API Keys (Optional):
    • Google Fact Check Tools API key
    • NewsAPI key
5. Test the Extension

Send yourself this test message on Telegram Web:

URGENT!!! BREAKING NEWS!!! Scientists HATE this one WEIRD TRICK! They don't want you to KNOW this SHOCKING truth!!! SHARE NOW!!!

Expected: 🚫 High risk warning (red badge)

Installation Complete! Your extension is now active and protecting you from misinformation.

Detection Capabilities

The extension detects:

  • Suspicious Keywords: "URGENT", "SHOCKING", "they don't want you to know"
  • Excessive Capitalization: MORE THAN 30% IN CAPS
  • Excessive Punctuation: Multiple exclamation marks!!!
  • Emotional Manipulation: Fear, anger, or excitement-based language
  • Suspicious URLs: Shortened links (bit.ly, tinyurl) and unreliable domains
  • Missing Sources: Claims without credible citations

Understanding Risk Levels

ℹ️ Low Risk (0-39%)

Minor concerns detected. Apply normal critical thinking. Generally safe to read but verify before sharing widely.

⚠️ Medium Risk (40-69%)

Multiple warning signs present. Verify from credible sources before trusting or sharing. Exercise increased caution.

🚫 High Risk (70-100%)

Strong indicators of misinformation. Likely false or misleading. Do not share. Investigate thoroughly if information seems important.

Important: This tool provides guidance, not absolute truth. Always use your own judgment and verify important information from multiple credible sources.
