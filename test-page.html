📋 How to Use This Test Page

⚠️ Important: This extension only works on web.telegram.org. This test page is for documentation and understanding the extension's behavior.

Installation Steps:

  1. Generate icon files (see INSTALLATION.md)
  2. Load extension in Chrome: chrome://extensions/
  3. Enable Developer Mode (toggle in top-right)
  4. Click "Load unpacked" and select hackomania2026 folder
  5. Navigate to web.telegram.org
  6. Open any chat and observe message analysis

🎯 Detection Criteria

The extension analyzes messages based on multiple factors:

🔍 Suspicious Keywords

Detects words like "breaking", "urgent", "shocking", "they don't want you to know", "miracle", "secret"

📢 Excessive Caps

Flags messages with more than 30% capital letters (SHOUTING)

❗ Punctuation Abuse

Identifies excessive use of exclamation or question marks (!!!, ???)

😱 Emotional Language

Detects emotionally manipulative words designed to provoke strong reactions

🔗 Suspicious URLs

Flags shortened URLs (bit.ly, tinyurl) and unreliable domains

📰 Missing Sources

Identifies claims without credible source citations or references

⚙️ Features

✅ Real-time Analysis: Messages are analyzed as they appear in your chats
✅ Visual Indicators: Color-coded warnings (🚫 High, ⚠️ Medium, ℹ️ Low)
✅ Detailed Explanations: Click "Details" to see why a message was flagged
✅ External Verification: Optional fact-checking with Google and NewsAPI
✅ Two Modes: Automatic (continuous) or Manual (on-demand) checking
✅ Privacy-Focused: All analysis done locally, no data collection

📊 Expected Results

High Risk Messages (70-100%):

  • Multiple suspicious keywords (5+ matches)
  • Heavy use of ALL CAPS
  • Excessive punctuation (!!!, ???)
  • Emotional manipulation words
  • Shortened or suspicious URLs
  • Claims without sources
  • Result: Red 🚫 warning badge

Medium Risk Messages (40-69%):

  • Some suspicious keywords (2-4 matches)
  • Moderate sensationalism
  • Some emotional language
  • Possible source issues
  • Result: Orange ⚠️ warning badge

Low Risk Messages (0-39%):

  • Minor concerns (1-2 warning signs)
  • Mostly neutral language
  • May include credible sources
  • Result: Yellow ℹ️ info badge or no warning

🔧 Configuration

Click the extension icon in your Chrome toolbar to access:

Basic Settings:

  • Enable Protection: Toggle extension on/off
  • Checking Mode: Automatic (continuous) or Manual (on-demand)

API Integration (Optional):

  • Google Fact Check Tools API: Verify claims against known fact-checks
  • NewsAPI: Check if claims appear in mainstream news

Statistics:

  • Messages Checked: Total analyzed
  • Messages Flagged: Total with warnings

⚠️ Important Notes

Not Perfect: This tool provides guidance, not absolute truth. Always verify important information from multiple credible sources.
False Positives: Some legitimate messages may be flagged due to language patterns. Use your judgment.
False Negatives: Sophisticated misinformation may not be detected. Stay critical!
Privacy: All analysis is performed locally in your browser. No data is collected unless you use external API features.
